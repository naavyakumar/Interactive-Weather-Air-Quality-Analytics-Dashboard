# Interactive-Weather-Air-Quality-Analytics-Dashboard
This Power BI dashboard presents city-level weather and air quality insights, including current conditions, 7-day temperature trends, humidity, wind speed, pressure, UV index, precipitation, and chance of rain. It also displays sunrise/sunset timings and AQI with pollutant breakdown, using interactive slicers and a clean dark UI for easy analysis.
